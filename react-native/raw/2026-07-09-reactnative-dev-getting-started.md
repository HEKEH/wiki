## What is React Native?

React Native is a framework for building native mobile applications using JavaScript and React. It enables developers from various backgrounds (web, Android, iOS, beginners) to create cross-platform mobile apps that render as native components on iOS and Android.

## Core Mental Model

- **Native Components**: React Native maps to actual native UI components rather than web equivalents.
- **JavaScript-based**: Uses JavaScript fundamentals with React patterns.
- **Cross-platform**: Write once, target multiple platforms (iOS, Android).
- **Inclusivity**: Documentation addresses developers of all experience levels.
