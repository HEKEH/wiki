# 官方文档:Introduction / Getting Started

React Native 官方文档的入口页,定义了框架的心智模型与学习主线。是整个知识库的**基础主线来源**。

## 核心心智模型

React Native 用 JavaScript + React 构建**真正的原生应用**,而非 WebView:JSX 组件映射到 iOS/Android 的原生 UI 组件,而不是 Web 元素。一次编写,多端(iOS、Android)渲染为各自平台的原生控件。

## 学习主线(The Basics)

官方推荐的线性路径:

1. Introduction — 概览与环境
2. **Core Components and Native Components** — RN UI 的基石
3. React Fundamentals — 必要的 React 概念
4. Handling Text Input — 用户输入
5. Using a ScrollView — 滚动内容
6. Using List Views — 高效列表渲染
7. Troubleshooting — 常见问题
8. Platform-Specific Code — iOS/Android 差异
9. More Resources — 延伸材料

## 进阶主题(Advanced)

Environment Setup、Workflow & Running on Device、UI & Interaction / Styling、Debugging & Testing、**Performance Optimization**、JavaScript Runtime、**Codegen**、Native Development、Android/iOS 平台指南、Legacy Architecture。

> 进阶部分中的 Codegen、Native Development、Performance 与新架构强相关,详见 [[concepts/01-new-architecture]]。

## 学习方式建议

- 先具备 JavaScript 基础。
- 用内嵌的 **Snack Player** 在浏览器里边读边写,再迁移到本地环境。
